# :hammer: Funcionalidades do sistema

- `Autenticação de usuário`: permitindo que os usuários acessem suas contas através de login e senha;
- `Exibição de saldo`: mostrando o saldo atual disponível na conta do usuário;
- `Depósito`: permitindo que o usuário faça depósitos em sua conta;
- `Saque`: permitindo que o usuário retire dinheiro de sua conta;
- `Transferência para outro usuário`: permitindo que o usuário transfira dinheiro de sua conta para outra conta de usuário cadastrado;
- `Exibição de histórico de transações`: permitindo que o usuário visualize um histórico de transações detalhado, incluindo datas, valores e descrições de cada transação.
